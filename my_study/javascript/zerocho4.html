객체 - 배열

배열 기본

객체(object): 객체는 자료형의 일종으로 다양한 값을 모아둔 또다른 값이다. 객체의 종류는 크게 배열(array), 함수(function), 배열이나 함수가 아닌 객체로 나눌 수 있다.

배열: 다양한 값, 다양한 자료형들을 하나로 묶어놓은 것

ex)
const apple = '사과';
const orange = '오렌지';
const pear = '배';
const strawberry = ' 딸기';
과일 종류가 매우 많기 때문에 모든 과일에 변수 이름을 붙이는 것은 고통스러운 일이다.
이런 경우 배열을 사용해 값들을 하나로 묶을 수 있다.
ex) const fruits = ['사과', '오렌지', '배', '딸기'];
상수 fruits에 담긴 사과, 오렌지, 배, 딸기 등 각각의 값을 "배열 안에 담긴 요소(element)"라고 칭한다.

배열을 만들려면 대괄호 [] 로 값들을 감싸주면 된다. 값은 쉼표로 구분한다.

배열 내부의 값을 개별적으로 불러올 수도 있다. 배열 이름 뒤에 불러오고 싶은 값의 자릿수를 적어주면 된다. (0부터 시작)
ex) const fruits = ['사과', '오렌지', '배', '딸기'];
console.log(fruits[0]); //사과
console.log(fruits[1]); //오렌지
console.log(fruits[2]); //배
console.log(fruits[3]); //딸기

프로그래밍에서는 자릿수를 index라고 부른다. ex) 0번째 인덱스...

배열 안에 있는 값이 꼭 문자열이지 않아도 되며, 값의 자료형이 모두 같지 않아도 된다.(함수도 가능)
배열 안에 다른 배열이나 변수를 넣을 수도 있다.
ex)
const arrayOfArray = [[1, 2, 3], [4, 5]]; //arrayOfArray라는 배열 안에 또 배열이 있음.
console.log(arrayOfArray[0]); // [1, 2, 3]
arrayOfArray처럼 배열 내부에 배열이 들어있는 경우, 내부에 있는 배열을 이차원 배열이라고 한다.

ex)
const a = 10;
const b = 20;
const variableArray = [a, b, 30];
console.log(variableArray[1]); // 20 (b의 값)

ex)
const everything = ['사과', 1, undefined, true, '배열', null];
const duplicated = ['가', '가', '가', '가', '가'];
const empty = []; //아무 값이 없는 배열
배열 내부의 값은 중복되어도 되고, 아무 값이 없는 배열도 만들 수 있다.

배열의 요소 개수 구하기

프로그래밍으로 요소의 개수를 구할 수도 있다.
앞에서 만든 everything 배열의 요소 개수를 구해보자.
배열 이름 뒤에 length를 붙이면 된다.

const everything = ['사과', 1, undefined, true, '배열', null];
console.log(everything.length) //6
빈 값들도 length에 당연히 포함된다.

배열의 요소 개수를 활용해 원하는 인덱스의 요소를 찾을 수 있다.
배열의 요소 개수는 항상 마지막 인덱스보다 1이 크다.
everything 배열에서 null의 인덱스는 5이고, 요소의 개수는 6이다.
즉, 배열의 요소 개수에서 1을 빼면 마지막 요소의 인덱스가 된다. 이를 활용해 마지막 요소의 값을 찾아낼 수 있다.

const findLastElement = ['a', 'd', 'c', 'd', 'e'];
findLastElement[findLastElement.length - 1];
설명: findLastElement[0]은 배열의 첫번째 값을 갖고 온다.
'findLastElement의 배열 개수 - 1'을 해서 나온 숫자가 대괄호 안에 들어가면서 마지막 요소를 불러오는 것이다.

배열에 요소 추가, 마지막에 추가, 요소 수정, 첫번째 앞 요소 추가

배열에 요소 추가하기

배열을 만든 후, 중간에 배열을 수정할 수 있다. 배열에 요소를 추가할 수도 있고, 특정 인덱스의 요소를 수정할 수도 있으며, 제거할 수도 있다.

const target = ['a', 'b', 'c', 'd', 'e'];
target[5] = 'f'
console.log(target) //['a', 'b', 'c', 'd', 'e', 'f']

배열의 마지막에 요소 추가하기

target 배열의 요소 중 e의 인덱스가 4이므로 target[5]에 f를 대입하면 e보다 뒤에 f가 위치한다.
배열의 마지막 요소 인덱스가 배열.length - 1 이므로 다음 요소를 추가하려면 배열.length에 값을 넣으며 된다.
ex) const target = ['가', '나', '다', '라', '마'];
target[target.length] = '바';
console.log(target) //['가', '나', '다', '라', '마', '바'];

또는 target.push('추가할 요소'); 로도 가능하다.

배열의 요소 수정하기

const target = ['나', '다', '라', '마'];
target[0] = '가';
console.log(target); //['가', '다', '라', '마'];
추가하는 방법과 비슷하다.

배열의 첫번째 앞에 요소 추가하기

배열 제일 앞에 요소를 추가할 때는 unshift라는 기능 사용

const target = ['나', '다', '라', '마', '바'];
target.unshift('가');
console.log(target); //['가', '나', '다', '라', '마', '바'];

상수 안 배열이 위 같은 방법들로 수정 가능한 이유

const에느 새로운 값을 대입(=)하지만 못한다고 기억하는 게 맞다.
const에 객체(배열, 함수, 객체 리터럴)가 대입된 경우 객체 내부의 속성이나 배열의 요소는 수정할 수 있다.
const target2 = ['a', 'b', 'c', 'd', 'e'];
target2 = ['f', 'g']; //불가능
target[0] = 'h' //가능

수정

마지막 요소 제거하기

마지막 요소를 제거할 때는 .pop();을 사용한다.
ex)
const target = ['가', '나', '다', '라', '마'];
target.pop(); //괄호에는 아무것도 안 들어감
console.log(target); //['가', '나', '다', '라'];

첫번째 요소 제거하기

첫번째 요소를 제거할 때는 .shift();를 사용한다.
ex)
const target = ['가', '나', '다', '라', '마'];
target.pop(); //괄호에는 아무것도 안 들어감
console.log(target); //['가', '나', '다', '라'];

중간 요소 제거하기

중간 요소를 제거할 때는 .splice(index, index로부터 지울 요소의 개수)를 사용한다.
ex)
const target = ['가', '나', '다', '라', '마'];
target.splice(1, 2);
console.log(target); //['가', '라', '마'];

.splice(index, n)을 사용할 때 (index)만 적는다면, 해당 인덱스로부터 뒤에 있는 값은 모두 지워진다.
ex)
const target = ['가', '나', '다', '라', '마'];
target.splice(2);
console.log(target); //['가', '나'];

중간 요소 제거하고 추가하기

중간 요소를 제거하고 추가할 때는.splice(index, n, 추가할 요소, 추가할 요소)를 사용한다.
지운 자리에 껴넣는다고 생각하면 된다.
ex)
const target = ['가', '나', '다', '라', '마'];
target.splice(0, 2, '추가1', '추가2', undefined);
console.log(target); //['추가1', '추가2', undefined, '다', '라', '마'];

중간 요소 제거하지 않고 추가하기 = 중간에 요소 추가하기

중간 요소를 단수 추가할 때는.splice(index, 0, 추가할 요소, 추가할 요소)를 사용한다.
ex)
const target = ['가', '나', '다', '라', '마'];
target.splice(1, 0, '추가');
console.log(target); //['가','추가, '나', '다', '라', '마'];

조회

배열에 찾는 요소가 있는지 없는지 조회

배열에 특정 요소가 있는지 찾을 수 있다.
일종의 검색 기능이라고 보면 된다. includes를 사용한다.

찾는 요소가 있으면 true, 없으면 false가 나온다.

ex)
const target = ['가', '나', '다', '라', '마']; const result1 = target.includes('다'); const result2 = target.includes('카'); console.log(result1) //true console.log(result2) //false

배열에 찾는 요소가 몇 번째에 있는지 조회

검색하고 싶은 값이 몇 번째 index에 위치하는지도 알 수 있다.
indexOf와 lastIndexOf를 사용한다.

주의할 점: 중복된 값이 있을 수 있기 때문에, 앞에서부터 찾을지, 뒤에서부터 찾을지가 중요한 것 같다.

ex)
const target = ['라', '나', '다', '라', '다'];
const result1 = target.indexOf('다'); //index 0부터 몇 번째에 있는지 찾아줌
const result2 = target.lastIndexOf('라'); //마지막 index부터 몇 번째에 있는지 찾아줌
const result3 = target.indexOf('가'); //찾는 값이 배열에 없으면 -1이 나온다.

console.log(result1) //2
console.log(result2) //마지막 index값이 4여서 3이 나온다.
console.log(result3) //-1
